# Pseudo-code-vers-C
Ce projet consiste à concevoir et implémenter un compilateur pour un langage de pseudo-code algorithmique en français. Il a pour objectif de transformer un programme compréhensible par l’humain en un code exécutable par la machine, tout en facilitant l’apprentissage de la logique algorithmique.

Le compilateur repose sur une définition formelle du langage (alphabet et règles grammaticales) et met en œuvre les principales phases de la compilation : analyse lexicale, syntaxique et sémantique. Lorsque le programme source est correct, une génération automatique de code en langage C est réalisée.

Ce projet vise à illustrer de manière concrète les concepts fondamentaux de la compilation à travers le développement d’un outil pédagogique fonctionnel.
